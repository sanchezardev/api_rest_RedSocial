
**UsuarioController (/usuario)**

* GET /usuario/todos → Obtiene todos los usuarios
* GET /usuario/{id} → Obtiene un usuario por su ID
* POST /usuario → Crea un nuevo usuario
* DELETE /usuario/{id} → Elimina un usuario por su ID

**PerfilController (/perfil)**

* GET /perfil/todos → Obtiene todos los perfiles
* GET /perfil/{id} → Obtiene un perfil por su ID
* POST /perfil → Crea un nuevo perfil
* DELETE /perfil/{id} → Elimina un perfil por su ID

* POST /comentario → Crea un nuevo comentario
